- HTTP
- HTML
- WWW
- Web Browser
- Web Server
- Web Page

Berners Lee pula menjadi direktur dari organisasi W3C, Bertugas untuk :
- Menciptakan standar dan aturan untuk Web
- Menciptakan tag-tag baru atau penghapusan untuk HTML,CSS DLL

- CSS, Diibaratkan sebagai baju dll untuk menghiasi kepada manequin
